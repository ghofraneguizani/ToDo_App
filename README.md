# ToDoList


Pour utiliser l'application, installer les modules suivants:

Express,
MongoDB,
Bcrypt,
jsonwebToken,
jwt-utils.

Pour les installer, il suffit de se placer dans le répertoire du projet et d'éxécuter la commande ci-dessous :

npm install
Puis lancer le serveur :

npm start
